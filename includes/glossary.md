# Base de données des termes DECADE-X (Ne pas publier directement)

--8<-- [start:def_api]
**API (Application Programming Interface) :** Interface informatique qui permet à différents logiciels de communiquer entre eux de manière sécurisée et standardisée.
--8<-- [end:def_api]

--8<-- [start:def_v2x]
**V2X (Vehicle-to-Everything) :** Système de communication permettant à un véhicule d'échanger des données avec son environnement (autres véhicules, infrastructures, piétons).
--8<-- [end:def_v2x]

--8<-- [start:def_adms]
**ADMS :** Architecture Data Management System, le standard de structuration des métadonnées pour notre catalogue.
--8<-- [end:def_adms]
